# MathRush

MathRush adalah aplikasi pembelajaran matematika berbasis mobile yang menggabungkan konsep game (gamifikasi) agar belajar lebih seru.

Pengguna dapat:
- Mengerjakan soal matematika
- Mendapatkan poin dan reward
- Naik level
- Melihat leaderboard
- Belajar bersama AI Tutor

Aplikasi ini dirancang agar proses belajar matematika menjadi lebih interaktif, menyenangkan, dan tidak membosankan.
