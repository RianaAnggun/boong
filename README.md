# MathRush
MathRush adalah aplikasi pembelajaran matematika berbasis mobile yang menggabungkan konsep game (gamifikasi) agar belajar lebih seru. Pengguna mengerjakan soal, mendapatkan poin, naik level, dan reward. Dilengkapi fitur quiz, leaderboard, dan AI tutor untuk membantu belajar secara interaktif dan menyenangkan.
